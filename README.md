# AI Fact Checking Web App

App Link - https://github.com/khushigaur01/fact-checker-app/edit/main/README.md


## Features
- Upload PDF
- Extract claims automatically
- Verify claims using live web data
- Detect fake or outdated information
- Streamlit deployment

## Tech Stack
- Streamlit
- PyMuPDF
- DuckDuckGo Search
- Python

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
